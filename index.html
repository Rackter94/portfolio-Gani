About Me

Assalamualaikum Wr.Wb, Perkenalkan Saya Gani dari tasikmalaya. Saya seorang pengembang Junior Node JS dengan hasrat yang kuat untuk mempelajari hal-hal baru. Saya akrab dengan beberapa kerangka kerja Node JS, dan saya telah mengembangkan API untuk sistem produksi menggunakan Express. Saat ini saya tertarik dan belajar tentang blockchain.
